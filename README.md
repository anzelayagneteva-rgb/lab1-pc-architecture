# Лабораторная работа №1
## Исследование архитектуры ПК

### 📋 Описание
Интерактивный веб-отчёт по лабораторной работе, посвящённой изучению основных компонентов персонального компьютера и определению их характеристик.

### 🎯 Цель работы
Изучить основные компоненты персонального компьютера и определить их характеристики.

### 📦 Содержание отчёта
- ✅ Цель работы
- ✅ Оборудование и программное обеспечение
- ✅ Порядок выполнения (6 шагов)
- ✅ Таблица характеристик компьютера (CPU, RAM, SSD, GPU)
- ✅ Структурная схема архитектуры ПК
- ✅ Выводы

### 🛠️ Используемые инструменты
- Диспетчер задач Windows (вкладка «Производительность»)
- CPU-Z
- AIDA64

### 🚀 Технологии
- React 19
- TypeScript
- Vite
- Tailwind CSS 4

### 📥 Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

### 📸 Превью
Отчёт включает:
- Интерактивную таблицу характеристик ПК
- Визуальную схему архитектуры фон Неймана
- Адаптивный дизайн для всех устройств
